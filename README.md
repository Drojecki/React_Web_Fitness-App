**Steps to Run the Application**


**Clone the Repository:**
```bash
git clone https://github.com/Drojecki/React_Web_Fitness-App.git
cd React_Web_Fitness-App
```

**Run the Application:**
  ```bash
docker-compose up
   ```

**Acces the application:**

Frontend: Open http://localhost:3000 in your browser to view the React frontend.

Backend API: The backend API runs on http://localhost:5000.

**Troubleshooting**
If you encounter issues on the first run, try:

```bash
docker-compose down
docker-compose up --build
```
